# Day Tracker Bot

Личный Telegram-бот для ежедневного трекинга дня и вечернего самоанализа.

## Возможности
- Трекинг дня по 7 категориям
- Оценка дня от 1 до 10
- Заметки к дню
- Рекомендации на завтра (без AI API)
- Статистика за неделю и месяц
- Экспорт данных в CSV и JSON
- Вечерние напоминания

## Установка
1. Установить Python 3.10+
2. python -m venv .venv
3. .venv\Scripts\Activate.ps1
4. pip install -r requirements.txt
5. Скопировать .env.example в .env и заполнить
6. python bot.py
